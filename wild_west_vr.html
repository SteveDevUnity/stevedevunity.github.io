Wild West VR

Back to Projects

Overview

Wild West VR is a high-performance 3D demo, optimized for 90 FPS on Meta Quest 3. Players experience an immersive western town in VR with their own avatar, interactive tutorial, scalable level system and permanently saveable progress. I developed the full experience independently, creating core systems and gameplay logic myself, while customizing and integrating some external models for optimal performance and visual consistency.

Features

  • Scalable level system (via ScriptableObjects)
  • Multilingual support (4 languages)
  • Persistent save system (highscores, player stats)
  • Performance optimized for Meta Quest 3
  • Custom 3D assets (Blender + Substance Painter)
  • Interactive dialogue & tutorial system
  • Stylized environment design
